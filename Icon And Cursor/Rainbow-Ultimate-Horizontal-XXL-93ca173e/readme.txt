Rainbow Ultimate Horizontal XXL
===============================

Author: Hopachi

This animated cursor set contains 20 cursors.

Files included:
  - ns.ani
  - nwse.ani
  - neutral.ani
  - pen.ani
  - pendulum.ani
  - text.ani
  - no.ani
  - up.ani
  - person.ani
  - alt_arrow.ani
  - arrow.ani
  - cross.ani
  - working.ani
  - busy.ani
  - ew.ani
  - link.ani
  - help.ani
  - location.ani
  - move.ani
  - nesw.ani

Installation:
  1. Right-click install.inf
  2. Select 'Install'
  3. Open Control Panel > Mouse > Pointers
  4. Select 'Rainbow Ultimate Horizontal XXL' from the Scheme dropdown
